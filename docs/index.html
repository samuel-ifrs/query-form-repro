HTTP QUERY: <form> vs fetch() — live demo

Repro for whatwg/html#12594. Each form submits to httpbingo.org/anything, which echoes back the HTTP method it actually received. No install required.

1. HTML forms (no JavaScript)

Three identical forms; only the method attribute differs. After you click, look at the "method" field in the JSON the echo server returns.

2. fetch() for comparison

(result appears here)
Expected result: the method="query" form makes the echo server report "method": "GET" (its body dropped), because HTML only defines get/post as form methods and an unknown value falls back to GET. The fetch() call, by contrast, reports "method": "QUERY". This is the gap #12594 is about.
